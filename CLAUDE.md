# VibeGame Studio - 游戏策划案智能生成

本项目是一个基于 Claude Code 的游戏策划案智能生成工具。

## 项目说明

这是一个多 Agent 协作系统，用于将模糊的游戏创意转化为完整的策划案文档。

## 知识库

在生成策划案时，请参考 `docs/` 目录下的文档：
- `docs/my-designs/` - 历史策划案（作为设计风格参考）
- `docs/references/` - 行业参考资料（作为最佳实践参考）

## 输出规范

生成的策划案应保存到 `output/` 目录，格式为 Markdown。

## Agent 角色定义

本项目包含以下 Agent 角色（通过 /skill 调用）：

### /producer - 制作人
负责提炼核心玩法支柱和游戏愿景

### /system-designer - 系统策划
负责设计游戏系统架构和核心循环

### /lead-designer - 主策划
负责 Review 整体设计并输出最终策划案

## 使用方式

1. 输入游戏创意
2. 依次调用各 Agent 进行设计
3. 最终输出完整策划案

## 中文输出

请使用中文进行所有输出和文档编写。
